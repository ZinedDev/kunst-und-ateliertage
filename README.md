# Kunst- und Ateliertage 2026

A modern React application for the **Kunst- und Ateliertage 2026** event. This project serves as the official platform to provide information for visitors and contact details for the art and studio days happening on September 19–20, 2026.

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Motion (motion/react)
- **Routing:** React Router
